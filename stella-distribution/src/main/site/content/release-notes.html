Funcionalidades Implementadas (Release Notes)

Esta lista contém as funcionalidades implementadas no Caelum Stella.

Core

  • Validador: CPF
  • Validador: CNPJ
  • Validador: NIT (PIS/PASEP/SUS)
  • Validador: Incrição Estadual (Para os seguintes estados: AL,AM,CE,DF,MG,MT,PR,PE,PI,RJ,RN,RO,RR,RS,SC,SP e SE)
  • Formatador: CPF
  • Formatador: CNPJ
  • Formatador: NIT (PIS/PASEP/SUS)

Stella Faces

  • Tags de validação
    • validateCPF
    • validateCNPJ
    • validateNIT
    • validateIE_SP

Stella Hibernate Validator

  • Validador: @CPF
  • Validador: @CNPJ
  • Validador: @NIT (PIS/PASEP/SUS)

Stella Boleto

  • Bancos
    • Banco do Brasil
  • Exportação
    • PDF
    • PNG - exportação gera um PNG grande. Necessário imprimir com Best to Fit, ou inserir em algum processador de texto
