O que é?

É um dos modulos do Caelum Stella, que tem como objetivo fornecer uma série de componentes do Java Server Faces, facilitando a vida dos desenvolvedores brasileiros que adotam esta tecnologia.

Os componentes são compatíveis com qualquer implementação do JSF e podem ser usados com Facelets.

Validando Documentos com Stella-Faces

Uma das características mais fortes do Caelum Stella-Faces é a validação com agilidade e simplicidade. Veja os exemplos de validação com JSP, XHTML (do Facelets) e repare a simplicidade de uso.
