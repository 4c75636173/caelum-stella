O que é?

É um dos modulos do Caelum-Stella, que tem como função a validação por meio das tecnologias JSF e Facelets.

Validando Documentos com Stella-Faces

Uma das características mais fortes do Caelum Stella-Faces é a poderosa validação com agilidade e simplicidade. Veja os exemplos de validação com JSP e XHTML, e repare a simplicidade de uso.

Caelum Stella-Faces possuí uma CustomTag que valida os documentos. Dessa forma, melhora o código e agilidade de desenvolvimento.

O Caelum Stella-Faces facilita a validação de documentos utilizando as tecnologias Java Server Faces e Facelets.
