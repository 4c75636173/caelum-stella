Validando Documentos com Stella-Faces

O Caelum Stella-Faces foi desenvolvido para realizar validações em camadas de apresentação baseadas em Java Server Faces.

Veja os exemplos de validação e note a simplicidade dos codigos implementados.

Caelum Stella Faces possuí uma CustomTag que valida os documentos. Dessa forma, o desenvolvimento é simplificado e a matendo a clareza do código.

Partindo de uma aplicação configurada para JSF podemos começar a utilizar nosso validador na página.
